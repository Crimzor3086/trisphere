# TriSphere

A Next.js + TypeScript prototype for a premium intelligence platform.

## What is included

- Landing page with hero, live intelligence feed, personalized role selection, and trend overview.
- Core navigation for Home, Trends, Champions, Matches, and Copilot.
- Dark, glassmorphism-first design with polished data cards.

## Getting started

```bash
cd /home/junior/projects/Trisphere
npm install
npm run dev
```

Open `http://localhost:3000`.
